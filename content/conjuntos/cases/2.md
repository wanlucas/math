## 1 - Verdadeiro ou falso <br>
A = { 0, 3, 5 } <br>

Afirmação | Resposta
--------- | --------
5 ∈ A     | V
3 ⊂ A     | F
∅ ∈ A     | F
0 ∈ A     | V
∅ ⊂ A     | V
0 = ∅     | F
0 ∈ ∅     | F
0 ⊂ A     | F

explicação de 3 ⊂ A e 0 ⊂ A: 3 e 0 não são conjuntos

B ⊂ D     | V
A c B     | F

A = { a, b, c }

Afirmação     | Resposta
--------------| --------
c ∉ A         | F
{c} ∈ A       | F
{a, c} ⊂ A    | V
{a, b} ∈ A    | F
{b} ⊂ A       | V
{a, b, c} ⊂ A | V


Afirmação         | Resposta
----------------- | --------
∅ = { x tal que x ≠ x } | V
∅ ⊂ {∅}           | V
∅ ∈ {∅}           | V
∅ ⊂ ∅             | V

explicação de ∅ ⊂ {∅}: não há valores em um conjunto vazio, então não é possível afirmar que algum valor do primeiro não existe no segundo.
