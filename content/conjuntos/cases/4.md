## 2 - Dados os conjuntos A = { a, b, c }, B = { a, c, d, e }, C = { c, d }, D = { a, d, e }, verdadeiro ou falso:

| Afirmação                   | Resposta |
| -------------               | -------- |
| A - B = { b }               | V        |
| B - C = { a, e }            | V        |
| D - B = { c }               | F        |
| ∁ca = ∅                     | F        |
| ∁∅b = { a, c, d, e }        | V        |
| ∁db = { c }                 | V        |
| (A ∩ B) - D = { a, d, e }   | F        |
| B - (A ∪ C) = { e }         | V        |
| (∁cd) ∪ (∁db) = { a, c, e } | V        |

**Explicação de Cca = ∅ (f), mesmo que C ⊄ A, a forma como o complementar é calculado não muda**
